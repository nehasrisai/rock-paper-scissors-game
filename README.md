# Rock Paper Scissors Game

This is a simple Rock Paper Scissors game that I built while learning JavaScript. The project helped me understand how JavaScript interacts with HTML and how to make web pages respond to user actions.

# Features

* Play Rock, Paper, Scissors against the computer
* Random computer move generation
* Display whether you win, lose, or tie
* Keep track of wins, losses, and ties
* Reset the score at any time
* Play using keyboard shortcuts

# Technologies Used

* HTML
* CSS
* JavaScript

# What I Learned

While building this project, I learned:

* How to use functions in JavaScript
* How to work with variables and conditionals
* How to update HTML elements using the DOM
* How to handle button clicks and keyboard events
* How to organize code into reusable functions

I created this project as part of my JavaScript learning journey. It helped me practice the fundamentals of web development and gain confidence in writing interactive JavaScript applications.
